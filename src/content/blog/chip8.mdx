CHIP-8 is the classic "first emulator" project. The spec is small enough
to hold in your head but covers the core ideas: a fetch-decode-execute
loop, memory-mapped I/O, and timed display refresh.

## Opcode dispatch

Try stepping through this short program — it loads two registers, adds
them, then computes BCD:

<OpcodeStepper
  client:visible
  opcodes={[
    { hex: "0x6005", desc: "V0 = 5" },
    { hex: "0x6103", desc: "V1 = 3" },
    { hex: "0x8014", desc: "V0 += V1 (V0 = 8)" },
    { hex: "0xF033", desc: "BCD of V0 → memory[I..I+2]" },
  ]}
/>

The `client:visible` directive only hydrates the React island when it
scrolls into view. Pages stay fast.
